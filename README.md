jquery eqInput
==============

A simple equation input with bracket completion and highlighting. 
Uses jquery. 
*Does not work for IE < 9*

_Usage:_
```html
<div id="eq-input"></div>
```
```javascript
$('#eq-input').eqInput();
```

To change the highlight color and styling
```javascript
$('#eq-input').eqInput({highlightClass: 'some-custom-class'})
```
